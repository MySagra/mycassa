MyCassa POS

Versione: 2.0.0

Tipo: Progressive Web App (PWA)

Descrizione: Sistema POS moderno per gestione ordini, stampa scontrini e sincronizzazione con API REST.

Caratteristiche

  • Interfaccia moderna con supporto dark mode
  • Gestione ordini con carrello dinamico
  • Stampa automatica su stampanti ESC/POS
  • Sincronizzazione con backend REST API
  • Filtri per categorie prodotti
  • Ricerca ordini precedenti
  • Funzionamento offline (PWA)
